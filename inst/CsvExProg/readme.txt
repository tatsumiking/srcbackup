プログラムファイル内のファイルについての説明
default.env　c:\usesprogram\csvex\csvex.env　ファイルのデフォルト定義
「default.env」は通常は存在しない
csvex.exeフォルダーの下に「fieldkey.env」を
１行目　コメント行
２行目　集計ファイル保存位置
３行目　UNISDB保存位置
４行目　出力ファイル名の後ろにつく名前（ファイル名の項目の３番目）
５行目　出力ファイルのCSVセパレータ、出力形式
６行目　ファンクション表示文字列（[,]カンマ区切り４項目）
７行目　0:間隔指定　1:時刻指定のフラグ
８行目　間隔指定時間リスト（[,]カンマ区切り、xx時間xx分で記述)
９行目　時刻指定リスト([,]カンマ区切り、xx時xx分で記述）
１０行目　選択された間隔指定時間(内部使用)
１１行目　計測開始年月日(内部使用)
１２行目　計測開始時分(内部使用)
１３行目　出力ID　0:ユーザーID 1:社員ID
１４行目　出力ファイル 0:個別ファイル出力 0:単一ファイル出力
１５行目　出力ファイル名の前につく名前（ファイル名の項目の１番目）
１６行目　出力ファイル名の中間につく名前（ファイル名の項目の２番目）、
　　　　　年日時分部分のため文字列フォーマット(yyyyMMddHHmmss)


fieldkey.env　CSVファイル、出力フィールド定義
「fieldkey.env」は通常は存在しない
csvex.exeフォルダーの下に「fieldkey.env」を作成し出力内容を変更。
「fieldkey.env」がない場合、例）の状態で設定されます
「fieldkey.env」の内容
１行目コメント行
２行目フィールド名
３行目出力フィールド
４行目出力フォーマットAnd桁数

例）
２行目　データ区分,打刻日,打刻時間,シフトコード,出退勤フラグ,ユーザID,例外コード,ターミナルNO
３行目　  ,%C_Date%,%C_Time%,  ,%L_Mode%,%L_UID%,  ,%L_TID%
４行目　2,yyyMMdd,HHmm,2,,s|-10,2,2

tEnterフィールドとコマンド名の関係
コマンドは%で囲む
コマンド名    フィールド名
C_Date        C_Date	// 年月日
C_Time        C_Time	// 時分秒
L_TID         L_TID		// ターミナル番号
L_UID         L_UID		// ユーザーID
C_Name        C_Name	// ユーザー名
L_Mode        L_Mode	// 出退勤フラグ
C_Unique      C_Unique	// 社員番号
になります

--------------------Vicsell版　fieldkey.envの例-------------------------
// 
データ区分,打刻日,打刻時間,シフトコード,出退勤フラグ,ユーザID,例外コード,ターミナルNO
  ,%C_Date%,%C_Time%,  ,%L_Mode%,%L_UID%,  ,%L_TID%
2,yyyMMdd,HHmm,2,,s|-10,2,2
------------------------------------------------------------

--------------------MajerFlow版 fieldkey.envの例-------------------------
// 
社員番号,西暦/月/日,時:分,出退勤フラグ,固定値
%C_Unique%,%C_Date%,%C_Time%,%L_Mode%,0
,yyyy/MM/dd,HH:mm,,
------------------------------------------------------------
